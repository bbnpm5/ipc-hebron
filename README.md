# IPC Hebron - Church Website

A modern, responsive website for India Pentecostal Church Hebron, Neria.

## Features

- Daily changing Bible verses
- Online giving/donations
- Sermon archive
- Event calendar
- Prayer request form
- Photo gallery
- Responsive design
- Modern UI/UX

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## Tech Stack

- **React 18.2.0** - Latest stable version
- **Vite 5.0.8** - Modern build tool
- **React Router v6** - Client-side routing
- **Tailwind CSS 3.3.6** - Utility-first CSS framework
- **i18next** - Internationalization (English, Kannada, Malayalam)
- **Lucide React** - Modern icon library
