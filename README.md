# IPC Hebron - Church Website

A modern, responsive website for India Pentecostal Church Hebron, Neria.

## Features

- Daily changing Bible verses
- Online giving/donations
- Sermon archive
- Event calendar
- Prayer request form
- Photo gallery
- Responsive design
- Modern UI/UX

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```
